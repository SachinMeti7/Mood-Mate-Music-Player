# Mood Music App

A MERN stack application for discovering songs based on your mood.

## Features
- Landing page
- Explore to home page
- Mood-based song recommendations
- Songs stored in MongoDB
- Clean UI

## Setup

### Backend
1. Navigate to `backend` directory
2. Install dependencies: `npm install`
3. Set up MongoDB and update `.env` with your MONGO_URI
4. Run the server: `npm start` or `npm run dev`

### Frontend
1. Navigate to `frontend` directory
2. Install dependencies: `npm install`
3. Start the React app: `npm start`

## Usage
- Visit the landing page
- Click "Explore" to go to home
- Select a mood to see recommended songs

## Troubleshooting
- Ensure MongoDB is running locally or update MONGO_URI
- Backend runs on port 5000, frontend on 3000
